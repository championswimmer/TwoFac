Migrate from Proton Authenticator to TwoFac

The safest way to migrate from Proton Authenticator to TwoFac today is to re-register each account one by one while keeping Proton installed until every login is verified.

Why the Manual Method Is the Safest Option

Proton Authenticator uses its own encrypted backup and restore flow. That is great for Proton-to-Proton restores, but it is not a direct import path for TwoFac today. Because of that, the most reliable migration method is the classic one: re-register each service with TwoFac.

It takes longer than importing a plain-text export, but it avoids format-conversion mistakes and guarantees that every service is explicitly enrolled with your new authenticator.

Step 1: Prepare Before Changing Anything

  1. Keep Proton Authenticator installed and working throughout the migration.
  2. Collect any recovery codes the service gives you before you modify 2FA settings.
  3. Make a shortlist of your most important accounts first: email, password manager, cloud storage, banking, GitHub, and work logins.
  4. For each account, sign in and open its Security or Two-Factor Authentication settings.

Step 2: Add TwoFac One Service at a Time

Different services behave differently, so use the safest option each site offers:

  1. If the service allows more than one authenticator app: add TwoFac as a second authenticator first, test it, and only then remove Proton from that account.
  2. If the service allows only one authenticator app: temporarily disable authenticator-app 2FA, re-enable it, and scan the newly issued QR code using TwoFac.
  3. Immediately test the new code before leaving the security page.

This slow-and-steady approach is especially important for email accounts and password managers, because those are the accounts you need to recover everything else.

Step 3: Verify Everything Before Removing Proton

Do not uninstall Proton Authenticator until every important service has been tested with a live login using a code from TwoFac.

  • Enable Biometric Lock in TwoFac.
  • Create an Encrypted Backup in TwoFac once your accounts are migrated.
  • Store new recovery codes securely if any service regenerated them during re-enrollment.

Step 4: Remove Proton Authenticator Only When Ready

Once you have successfully used TwoFac for all critical services across your devices, you can remove the entries from Proton Authenticator or uninstall it entirely. Until then, keeping Proton as a temporary fallback is the safest choice.
