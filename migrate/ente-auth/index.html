Migrate from Ente Auth to TwoFac

Migrating from Ente Auth to TwoFac is incredibly easy using the Ente Auth export functionality. TwoFac officially supports importing Ente Auth text exports.

Step 1: Export from Ente Auth

You can export your secrets using either the Ente Auth mobile or desktop apps.

  1. Open the Ente Auth app on your device.
  2. Open the Navigation Menu (three horizontal lines or "hamburger" icon) in the top-left corner.
  3. Select Settings (or Data).
  4. Click or tap on Export Codes.
  5. Choose the Plain Text Export format.
    • Why Plain Text? This generates an unencrypted .txt file containing a list of your secrets in standard otpauth:// URI format, which TwoFac uses.
  6. Save the file to a secure location on your device.

Security Warning: The unencrypted export file contains your raw 2FA secrets. Never share or upload this file to insecure cloud storage. Delete it permanently as soon as you are done migrating.

Step 2: Import into TwoFac

  1. Open the TwoFac app on your device.
  2. Navigate to Settings.
  3. Find the Import section.
  4. Select the plain text (.txt) file you exported from Ente Auth.
  5. TwoFac will parse the otpauth:// URIs and securely store all your accounts in your vault.

Step 3: Secure Your Setup

Because your secrets are now in TwoFac, ensure you take advantage of its security features:

  • Delete the .txt file: Make sure the plain text export file is permanently removed from your computer or phone.
  • Enable Biometric Lock: Secure TwoFac using fingerprint or face recognition so nobody but you can open it.
  • Set up an Encrypted Backup: Generate an encrypted backup directly from TwoFac so that your future backups are secured end-to-end with a passkey.

Step 4: Uninstall Ente Auth

After confirming that your codes are generating correctly in TwoFac, you can delete your Ente Auth account and uninstall the app.
